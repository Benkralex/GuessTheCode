So funktioniert's

Steuerung

Du kannst Zahlen/Farben einfach per Drag and Drop in das gewünschte Eingabefeld ziehen. Alternativ kannst du auch auf eine Zahl/Farbe tippen, um sie automatisch im nächsten verfügbaren Eingabefeld (von links nach rechts) einzufügen. Sobald alle Felder ausgefüllt sind, klicke auf "Prüfen".


Rückmeldung

In der Tabelle werden alle deine Versuche angezeigt. Ein grüner Haken ( check ) zeigt an, wie viele Zahlen/Farben an der richtigen Position stehen. Ein gelbes Ausrufezeichen ( warning ) zeigt an, wie viele Zahlen zwar im Code vorkommen, aber nicht an der richtigen Position sind.
Achtung: Es können Zahlen/Farben mehrfach im Code vorkommen.


Ziel

Dein Ziel ist es, den zufällig generierten Code anhand dieser Rückmeldungen zu erraten.


Musik

Über den Knopf rechts oben kann die von einer KI generierte Hintergrund-Musik aktiviert bzw. deaktiviert werden.
